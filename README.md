# CPO TARS (Chief Personal Officer - Totally Autonomous Reasoning System)

## Overview
CPO TARS is a Chrome Extension designed to be your personal AI assistant. It features:
- **Immersive Translation**: Translate content seamlessly.
- **Prompt Macros**: Trigger custom prompts with shortcuts.
- **VLM Screenshot**: Analyze screenshots using Gemini Vision API.

## Features
- **Translation**: High-quality translation using Gemini.
- **Macros**: Customizable prompt macros for quick actions.
- **Vision**: Understand what's on your screen.

## Development
1. Install dependencies: `pnpm install`
2. Run development server: `pnpm dev`
3. Load extension in Chrome: `dist/chrome-mv3-dev`
